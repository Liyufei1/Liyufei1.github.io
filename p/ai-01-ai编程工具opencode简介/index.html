简介

OpenCode 是一个开源的 AI 编程助手，它有以下几种运行方式: 终端命令行 、IDE插件 、独立桌面应用 、 云端环境 .相较于其他ai编程工具， OpenCode 具备以下几个优点：

  • 多平台支持: 支持终端界面、桌面应用和 IDE 扩展
  • 灵活的模型支持: 支持 75+ 个 LLM 提供商，包括 Claude、GPT、Gemini 等
  • LSP 集成: 自动加载适合 LLM 的语言服务器协议
  • 多会话: 可在同一项目上并行启动多个代理

安装

本文仅介绍最简单的终端命令行的安装方式，更多安装方式请查看 opencode文档

npm:

模型

OpenCode 支持连接到多种 LLM 提供商。你可以通过配置 API 密钥来使用不同的模型。

  1. 在 TUI 中运行 /connect 命令,选择模型供应商，并填写你的API Key
    1. (非必须) 一般使用 openrouter.ai 来对接模型,里面有许多免费模型的API,也有付费的.
    2. 也可以对接各种官方的模型,如OpenAI、google、anthropic
  2. 在 TUI 中运行 /model 命令,选择想使用的模型.
    1. 快捷键 F2 可以快速切换模型，反向切换快捷键 Shift+F2
    2. MiniMax-M2.1 : 编程能力较强
    3. GLM-4.7: 编程能力较强
  3. 对模型的配置，会持久化保存在 C:\Users\YourName\.local\share\opencode\auth.json当中，因此也可以通过对该文件的编辑，实现手动配置模型。

常用功能

基本命令

命令 说明
/init 初始化项目，创建 AGENTS.md
/connect 连接模型提供商
/model 切换使用的模型
/undo 撤销上次的更改（可多次撤销）
/redo 重做已撤销的更改
/compact 压缩对话，来释放模型上下文
/new 新建一个对话，可以并行运行
/session 查看所有对话
/timeline 可视化对话时间线，显示会话历史的树状结构，支持导航和分支
/share 生成分享链接，包含了整个对话记录（复制到剪贴板）
/unshare 取消分享
/export 将对话记录，导出成文件

工作模式（主Angent）

Plan 模式（规划模式）：

  • 按 Tab 键切换到规划模式
  • 在此模式下，OpenCode 不会直接修改代码，而是提供实现方案
  • 适合在开发前讨论和确认实现思路

Build 模式（构建模式）：

  • 再次按 Tab 键切换回构建模式
  • OpenCode 将直接执行代码修改
  • 适用于明确的开发任务

自定义 Agent 模式：

  • 按 Ctrl+Tab 键切换到自定义模式
  • 在~/.config/opencode/agent/your_agent.md中定义专属提示词
  • 可以为不同场景创建专门的 Agent（如代码审查、数据分析、前端开发等）
  • 格式规范请参考官方文档

SKILL和MCP

Agent Skills（智能体技能）

Agent Skills 允许你定义可重用的行为，OpenCode 可以通过原生 skill 工具按需加载这些技能。

Skill 存储位置

  • 项目级：.opencode/skills/<name>/SKILL.md
  • 全局：~/.config/opencode/skills/<name>/SKILL.md
  • Claude 兼容：.claude/skills/<name>/SKILL.md
  • Agents 兼容：.agents/skills/<name>/SKILL.md

MCP Servers（模型上下文协议）

MCP（Model Context Protocol）是一种让 AI 模型与外部工具和数据源交互的协议。OpenCode 支持 MCP 服务器，可以扩展其功能。

通过 MCP，你可以：

  • 连接数据库进行查询
  • 与版本控制系统集成
  • 访问外部 API
  • 使用自定义工具

配置 MCP 服务器在 ~/.config/opencode/opencode.json 的 mcps 部分。

云端环境

OpenCode 云服务

OpenCode 提供了云端环境支持，让你可以在远程服务器上使用 OpenCode。

主要特性

  • 远程开发：在远程服务器上运行 OpenCode
  • 协作功能：与团队成员共享会话
  • 共享链接：生成可分享的对话链接

生成分享链接

在任意会话中运行：

1

/share

这会将当前对话的链接复制到你的剪贴板。你可以将此链接分享给团队成员，以便：

  • 代码审查
  • 问题调试
  • 知识共享

注意：对话默认不公开分享，只有你主动分享后他人才可访问。

隐私与安全

OpenCode 采用隐私优先的设计：

  • 不存储你的代码或上下文数据
  • 所有数据处理在本地完成
  • 适合在隐私敏感的环境中使用

如需了解更多隐私政策，请访问 opencode.ai/docs/enterprise

总结

OpenCode 是一个功能强大、灵活且隐私友好的 AI 编程助手。无论你是个人开发者还是团队成员，它都能显著提升你的开发效率。

快速开始建议

  1. 安装 OpenCode：curl -fsSL https://opencode.ai/install | bash
  2. 配置你的 API 密钥
  3. 在项目目录运行 opencode 并执行 /init
  4. 开始与 AI 对话，探索代码库
